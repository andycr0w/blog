Windows 下常见的通用快捷键。

<!-- more -->

> 说明：以下内容主要面向 Windows 桌面环境，部分操作在不同版本界面名称略有差异，但核心功能基本一致。

## 文件与资源管理

| 快捷键 / 操作 | 助记 | 用途 |
| --- | --- | --- |
| `Win + E` | Explorer | 打开文件资源管理器。 |
| `Ctrl + Shift + N` | New | 新建文件夹。 |
| `F2` | 重命名 | 重命名当前选中的文件或文件夹。 |
| `Alt + Enter` | 属性 | 打开当前文件、文件夹或磁盘的属性窗口。 |
| `Alt + 双击文件` | 查看属性 | 用鼠标方式快速查看文件属性。 |
| `Ctrl + 双击文件夹` | 新窗口 | 用鼠标方式快速在新页面打开文件夹。 |
| `Ctrl + C` | Copy | 复制选中的文件或内容。 |
| `Ctrl + V` | Paste | 粘贴已复制的文件或内容。 |
| `Ctrl + X` | Cut | 剪切选中的文件或内容。 |
| `Delete` | 删除 | 删除选中的文件到回收站。 |
| `Shift + Delete` | `/` | 直接永久删除选中文件，使用前要确认无误。 |

## 窗口管理

| 快捷键 / 操作 | 助记 | 用途 |
| --- | --- | --- |
| `Win + ↑ / ↓` | 最大化 还原 最小化 | 最大化当前窗口 / 还原或最小化当前窗口。 |
| `Win + ← / →` | 靠左 靠右 | 将当前窗口贴靠到屏幕左侧 / 右侧。 |
| `Win + Shift + ←/→` | 跨屏移动 | 在多显示器之间移动当前窗口。 |
| `Win + D` | Desktop | 快速显示桌面，再按一次恢复窗口。 |
| `Win + M` | Minimize | 最小化所有窗口。 |
| `Win + Home` | 保留当前 | 除当前窗口外最小化其他所有窗口。 |

## 应用切换与任务操作

| 快捷键 / 操作 | 助记 | 用途 |
| --- | --- | --- |
| `Alt + Tab` | 切换窗口 | 在已打开应用窗口之间快速切换。 |
| `Alt + F4` | 关闭程序 | 关闭当前窗口或当前程序。 |
| `Win + Tab` | `/` | 打开任务视图，查看所有窗口和虚拟桌面。 |
| `Win + X` | 快速菜单 | 打开快速链接菜单，进入设备管理器、终端、任务管理器等系统入口。 |
| `Ctrl + Shift + Esc` | 任务管理器 | 直接打开任务管理器。 |
| `Ctrl + Alt + Delete` | `/` | 打开安全操作界面，可锁屏、切换用户或进入任务管理。 |

## 系统控制

| 快捷键 / 操作 | 助记 | 用途 |
| --- | --- | --- |
| `Win + L` | Lock | 立即锁定电脑。 |
| `Win + I` | Settings | 打开系统设置。 |
| `Win + R` | Run | 打开“运行”窗口。 |
| `Win + S` | Search | 打开系统搜索。 |
| `Win + V` | 剪贴板历史 | 查看剪贴板历史记录，并快速粘贴之前复制过的内容。 |
| `Win + A` | Action Center | 打开通知中心或快速设置面板。 |
| `Win + P` | Project | 切换投影或多屏显示模式。 |
| `Win + .` | 表情面板 | 打开表情、符号和颜文字面板。 |
| `Win + Space` | 切换输入法 | 切换输入法或键盘布局。 |

## 文本与截图

| 快捷键 / 操作 | 助记 | 用途 |
| --- | --- | --- |
| `Ctrl + A` | All | 全选当前内容。 |
| `Ctrl + Z / Y` | Undo Redo | 撤销上一步操作 / 恢复刚刚撤销的内容。 |
| `Ctrl + Backspace` | 删除前一词 | 删除光标前一个单词。 |
| `PrtSc` | Print Screen | 截取整个屏幕内容。 |
| `Alt + PrtSc` | 当前窗口截图 | 截取当前活动窗口。 |
| `Win + Shift + S` | Screenshot | 打开系统截图工具，选择区域截图。 |
