整理下 TIC-80 里常用的快捷键，分成系统全局快捷键、内置编辑器快捷键两部分，方便做游戏时快速查阅。

<!-- more -->

> 参考： [TIC-80 Manual & Cheat Sheet](https://skyelynwaddell.github.io/tic80-manual-cheatsheet/)

## 【系统全局快捷键】

| 快捷键 | 用途 |
| --- | --- |
| `Esc` | 在控制台和编辑器之间切换 |
| `Alt + ~` | 显示控制台 |
| `Alt + 1` / `F1` | 打开代码编辑器 |
| `Alt + 2` / `F2` | 打开精灵编辑器 |
| `Alt + 3` / `F3` | 打开地图编辑器 |
| `Alt + 4` / `F4` | 打开音效编辑器 |
| `Alt + 5` / `F5` | 打开音乐编辑器 |
| `Ctrl + PgUp / PgDn` | 切换到上一个 / 下一个编辑器模式 |
| `F11` / `Alt + Enter` | 切换全屏 / 窗口模式 |
| `Ctrl + R` / `Enter` | 运行当前项目 |
| `Ctrl + X / C / V` | 在编辑器中剪切 / 复制 / 粘贴 |
| `Ctrl + S` | 保存当前卡带 |
| `Ctrl + Z / Y` | 撤销 / 重做 |
| `方向键` | 导航移动 |
| `F6` | 切换 CRT 滤镜 |
| `F7` | 指定封面图 |
| `F8` | 截图 |
| `F9` | 开始 / 停止 GIF 录制 |

## 【内置编辑器快捷键】

### 代码编辑器

| 快捷键 | 用途 |
| --- | --- |
| `Ctrl + F` | 查找 |
| `Ctrl + G` | 跳转到指定行 |
| `Ctrl + Tab` | 缩进当前行 |
| `Ctrl + Shift + Tab` | 取消当前行缩进 |
| `Ctrl + O` | 显示代码大纲 |

### 精灵编辑器

| 快捷键 | 用途 |
| --- | --- |
| `[` / `]` | 选择上一个 / 下一个调色板颜色 |
| `-` / `=` | 调整画笔大小 |

### 地图编辑器

| 快捷键 | 用途 |
| --- | --- |
| `Shift` | 显示精灵表 |

### 音乐编辑器

| 快捷键 | 用途 |
| --- | --- |
| `Shift + Enter` | 从当前光标位置播放 pattern |

### 音效编辑器

| 快捷键 | 用途 |
| --- | --- |
| `Space` | 播放当前音效 |

## 使用建议

1. 最值得先记住的是 `Esc`、`Ctrl + R`、`Ctrl + S`、`Ctrl + Z / Y`。
2. 频繁在代码、精灵、地图之间切换，`Alt + 1~5` 和 `Ctrl + PgUp / PgDn` 会非常高频。
3. 做像素素材和调音时，精灵编辑器与音效编辑器这两组快捷键最省时间。
