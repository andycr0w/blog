浏览器里最常用的网页快捷键，按使用对象分类，并补上助记，方便更快形成肌肉记忆。

<!-- more -->

> 说明：下面以 Windows 下主流浏览器的通用操作为主，Chrome、Edge、Firefox 等常见浏览器大多都支持，少数快捷键可能略有差异。

## 标签页管理

| 快捷键 / 操作 | 助记 | 用途 |
| --- | --- | --- |
| `Ctrl + T` | Tab | 打开一个新标签页。 |
| `Ctrl + W` | 关闭标签页 | 关闭当前标签页。 |
| `Ctrl + Shift + T` | 重新打开 | 重新打开刚刚关闭的标签页。 |
| `Ctrl + Tab + (Shift)` | 标签切换 | 切换到右侧（左侧）标签页。 |
| `Ctrl + 1` 到 `Ctrl + 8` | 数字对应标签页序号。 | 快速跳转到指定序号的标签页。 |
| `Ctrl + 9` | 最后一个 | 快速跳转到最后一个标签页。 |
| `鼠标中键单击链接` | 中键新开 | 在新标签页中打开链接。 |
| `鼠标中键单击标签页` | 中键关闭 | 直接关闭当前标签页。 |
| `Ctrl + N` | New | 打开一个新的窗口。 |
| `Ctrl + Shift + N` | `/` | 打开一个新的无痕或隐私浏览窗口。 |

## 当前页面操作

| 快捷键 / 操作 | 助记 | 用途 |
| --- | --- | --- |
| `Ctrl + R` | Refresh | 刷新当前页面。 |
| `Ctrl + F5` | 强制刷新 | 强制刷新页面，忽略部分缓存重新加载。 |
| `Ctrl + F` | Find | 在当前页面内查找关键词。 |
| `Ctrl + P` | Print | 打印当前页面或导出当前页面为 PDF。 |
| `Ctrl + S` | Save | 保存当前网页。 |
| `Esc` | Escape | 停止当前页面继续加载。 |
| `空格` | 向下翻屏 | 向下滚动当前页面一屏。 |
| `Shift + 空格` | 向上翻屏 | 向上滚动当前页面一屏。 |
| `F11` | 全屏 | 在浏览器中切换全屏显示。 |

## 导航和地址栏

| 快捷键 / 操作 | 助记 | 用途 |
| --- | --- | --- |
| `Alt + ← / →` | 后退 前进 | 返回上一页 / 前进到下一页。 |
| `Ctrl + L` | Location bar | 选中地址栏，方便立即输入网址或关键词。 |
| `Ctrl + Enter` | 补全访问 | 在地址栏输入英文站点名后自动补全 `.com`。 |
| `Ctrl + D` | 收藏 | 收藏当前网页。 |
| `Ctrl + H` | History | 打开浏览历史记录。 |
| `Ctrl + J` | 下载记录 | 打开下载记录。 |
| `Ctrl + Shift + Delete` | 清除浏览数据 | 打开清除浏览数据界面。 |

## 页面滚动与定位

| 快捷键 / 操作 | 助记 | 用途 |
| --- | --- | --- |
| `Home / End` | 顶部 底部 | 快速回到当前页面顶部 / 底部。 |
| `Page Up / Page Down` | 上下翻页 | 向上 / 向下翻一页内容。 |
| `↑ / ↓` | 上下滚动 | 小幅向上或向下滚动页面。 |

## 缩放

| 快捷键 / 操作 | 助记 | 用途 |
| --- | --- | --- |
| `Ctrl + +` | 放大 | 放大页面或文档显示比例。 |
| `Ctrl + -` | 缩小 | 缩小页面或文档显示比例。 |
| `Ctrl + 0` | 默认比例 | 恢复默认缩放比例。 |
| `Ctrl + 鼠标滚轮` | 滚轮缩放 | 更直观地放大或缩小页面。 |

## 选择与输入

| 快捷键 / 操作 | 助记 | 用途 |
| --- | --- | --- |
| `Ctrl + C` | Copy | 复制选中内容。 |
| `Ctrl + V` | Paste | 粘贴剪贴板内容。 |
| `Ctrl + X` | Cut | 剪切选中内容。 |
| `Ctrl + Z` | Undo | 撤销上一步操作。 |
| `Ctrl + Y` | Redo | 恢复刚刚撤销的操作。 |
| `Ctrl + A` | All | 全选当前页面可选内容或当前输入区域内容。 |
