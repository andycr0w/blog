这篇整理 TUM EIKON Rechnerräume 机房里使用 KDE Plasma 时最常见的一组快捷键，优先覆盖机房里最常用的操作场景。

<!-- more -->

> 说明：KDE 官方文档里提到的 `Meta` 键，在大多数普通 PC 键盘上通常就是 `Win` 键。
>
> 参考： [KDE 官方文档：Common Keyboard Shortcuts](https://docs.kde.org/stable_kf6/en/khelpcenter/fundamentals/kbd.html)

## 打开与关闭程序

| 快捷键 | 用途 |
| --- | --- |
| `Meta` | 打开应用启动器 |
| `Alt + Space` / `Alt + F2` | 打开运行命令界面 |
| `Ctrl + Esc` | 打开系统活动 |
| `Alt + F4` | 关闭当前窗口 |
| `Ctrl + Q` | 退出当前程序 |
| `Ctrl + Alt + Esc` | 强制关闭程序 |

## 窗口切换与桌面浏览

| 快捷键 | 用途 |
| --- | --- |
| `Alt + Tab` / `Alt + Shift + Tab` | 在窗口间向前 / 向后切换 |
| `Ctrl + F10` | 显示所有窗口 |
| `Ctrl + F9` | 显示当前桌面的所有窗口 |
| `Ctrl + F7` | 只显示当前程序的窗口 |
| `Ctrl + F12` | 显示桌面 |
| `Alt + F3` | 打开窗口操作菜单 |
| `Ctrl + Alt + A` | 切换到正在请求注意的窗口 |

## 按方向切换窗口

| 快捷键 | 用途 |
| --- | --- |
| `Meta + Alt + ↑ / ↓` | 切换到上方 / 下方窗口 |
| `Meta + Alt + ← / →` | 切换到左侧 / 右侧窗口 |

## 缩放与窗口状态

| 快捷键 | 用途 |
| --- | --- |
| `Meta + =` | 放大桌面 |
| `Meta + -` | 缩小桌面 |
| `Meta + 0` | 恢复默认缩放 |
| `Meta + PgUp` | 最大化 / 恢复窗口 |
| `Meta + PgDn` | 最小化窗口 |

## 虚拟桌面与活动

| 快捷键 | 用途 |
| --- | --- |
| `Meta + Q` | 打开活动管理 |
| `Alt + D`，`Alt + A` | 打开活动管理 |
| `Meta + Tab` / `Meta + Shift + Tab` | 切换到下一个 / 上一个活动 |
| `Ctrl + F1 / F2 / F3 / F4` | 切换到桌面 1 / 2 / 3 / 4 |

## 桌面与组件

| 快捷键 | 用途 |
| --- | --- |
| `Alt + D`，`A` | 添加桌面组件 |
| `Alt + D`，`R` | 删除当前组件 |
| `Alt + D`，`L` | 锁定 / 解锁组件 |
| `Alt + D`，`S` | 打开组件设置 |
| `Alt + D`，`T` | 运行关联程序 |
| `Alt + D`，`Alt + S` | 打开桌面设置 |

## 文档与编辑

| 快捷键 | 用途 |
| --- | --- |
| `Ctrl + A` | 全选 |
| `Ctrl + Z` | 撤销 |
| `Ctrl + Shift + Z` | 重做 |
| `Ctrl + X / C / V` | 剪切 / 复制 / 粘贴 |
| `Ctrl + N` | 新建 |
| `Ctrl + P` | 打印 |
| `Ctrl + S` | 保存 |
| `Ctrl + F` | 查找 |
| `Ctrl + W` | 关闭文档或标签页 |
| `F5` | 刷新 |

## 文件管理

| 快捷键 | 用途 |
| --- | --- |
| `Ctrl + L` | 切换到路径输入 |
| `Ctrl + Shift + A` | 反选 |
| `Alt + ← / →` | 后退 / 前进 |
| `Alt + ↑` | 返回上级目录 |
| `Alt + Home` | 回到主目录 |
| `Delete` | 移到回收站 |
| `Shift + Delete` | 直接永久删除 |

## 帮助

| 快捷键 | 用途 |
| --- | --- |
| `F1` | 打开帮助 |
| `Shift + F1` | 查看当前元素说明 |

## 离开机房前常用

| 快捷键 | 用途 |
| --- | --- |
| `Ctrl + Alt + L` | 锁屏 |
| `Ctrl + Alt + Del` | 打开离开 / 注销 / 关机相关菜单 |

## 使用建议

1. 在机房里最值得先记住的是 `Meta`、`Alt + Tab`、`Alt + F4`、`Ctrl + Alt + L`。
2. 如果你经常同时开浏览器、终端、编辑器，`Meta + ←/→` 和 `Alt + Tab` 会很高频。
3. 离开座位前优先用 `Ctrl + Alt + L` 锁屏，比直接关窗口更稳妥。
4. 退出的时候直接 `Ctrl + Alt + Del` 即可。
